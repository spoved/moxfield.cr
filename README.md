# moxfield

An unofficial API for <https://www.moxfield.com>

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     moxfield:
       github: spoved/moxfield.cr
   ```

2. Run `shards install`
